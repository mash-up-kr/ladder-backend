# Ladder Server

### 개발 스택

- Language: Kotlin
- Framework: Spring Boot / Spring MVC / Spring Webflux
- Data: JPA / QueryDSL / MySQL 8.0 / Redis
- Test: Junit5 / Spring Test / RestAssured
- Socket: STOMP
- Infra: k8s
- Operation: Grafana / Prometheus
- CI/CD: Git / Github Actions / ArgoCD
